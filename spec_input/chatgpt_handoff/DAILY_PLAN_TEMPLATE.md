# DAILY_PLAN_TEMPLATE.md

## Purpose
Defines the exact daily Social Plan output format for Local Social.

## Standard Header
Use:

**TODAY: [Day] — [Area] Social Plan**

Example:
**TODAY: Thursday — Del Mar / Solana Beach Social Plan**

## If No Good Options Exist
Output exactly and prominently:

**No events today.**

Optional second sentence:
- No strong nearby social options matched today.

## If Options Exist
Use this structure:

**Primary Option:**  
[Venue] — [Exact time]  
- Live music / event: [lineup or event description]
- Crowd: [brief crowd description]

**Backup Option:**  
[Venue] — [Exact time]  
- Live music / event: [lineup or event description]
- Crowd: [brief crowd description]

Then include venue notes for applicable venues:

**Belly Up Tavern:**  
- [short relevance note]

**Monarch Ocean Pub:**  
- [short relevance note]

**Del Mar Plaza:**  
- [short relevance note]

If a venue is not relevant that day, say so briefly.

Then end with:

**Call:** GO / MAYBE / SKIP  
**Crowd Grade:** A / B / C  
**Priority Level:** High / Medium / Low  

**Note:**  
[short weather / turnout / practicality note]

## Tone
- concise
- readable
- no fluff
- useful in under 15 seconds of reading

## Example
**TODAY: Friday — Del Mar / Solana Beach Social Plan**

**Primary Option:**  
Monarch Ocean Pub — 4:00–7:00 PM  
- Live music: acoustic set  
- Crowd: conversational, repeat-friendly

**Backup Option:**  
Belly Up Tavern — 8:00 PM  
- Live music: 80s dance night  
- Crowd: higher energy, broader social mix

**Del Mar Plaza:**  
- No Seaside Sessions tonight

**Call:** GO  
**Crowd Grade:** A  
**Priority Level:** High  

**Note:**  
Warm weather should support good turnout and easy patio flow.
