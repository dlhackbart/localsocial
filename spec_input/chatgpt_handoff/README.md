# Local Social Add-On Package

This ZIP is an add-on for an existing Local Social Claude Code project.

It adds:
- reminder / notification behavior
- daily Social Plan formatting
- no-events logic
- nightly logging behavior
- terminology update from "dating plan" to "social plan"

## Files
- NOTIFICATION_SPEC.md
- LOGGING_SPEC.md
- DAILY_PLAN_TEMPLATE.md
- AUTOMATION_NOTES.md

## Recommended Claude instruction
Read these add-on files and integrate them into the existing Local Social project without replacing the current architecture.

Implement:
- 8:00 AM Social Plan reminder behavior
- 4:00 PM go-time reminder behavior
- "No events today." logic
- 9:30 PM nightly venue logging behavior
- Social Plan wording throughout
